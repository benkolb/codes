Misc.

Some pieces of code I wrote and find useful to have around.

Matlab

(S)VAR toolbox (work in progress) - so far it allows lag selection using AIC, Hannan-Quinn, BIC, Ljung-Box Q test and likelihood ratio test, and estimation of a VAR using OLS equation-by-equation. [to be done: Getting SVAR imposing different identification schemes]. Kolb.VAR.toolbox.zip


Notes and codes on Kalman filter (description of algorithm, derivations, two examples -- estimating the likelihood of a small New Keynesian DSGE model and estimating time-varying Taylor rule coefficients for US data): kalman.zip


Code for HP and bandpass (CF) filter: hpfilter_bk; bpfilter_bk


Code to make conditional forecasts in a VAR (conditional on a given course of up to two variables): condfc2.zip

Dynare

Minimal example of how to use a steady state file: steadystate.zip

"A Truly Baseline DSGE Model": Some codes implementing a stripped-down version of "A Baseline DSGE Model" by Jesús Fernández-Villaverde and Juan Rubio-Ramírez, allowing to switch on single features of their model (like sticky wages, capital utilisation, adjustment costs etc.) back on one by one. Comes with some notes. Truly_Baseline_DSGE.zip

R

Simple code to download data for a US Taylor rule from datamarket.com and save it to .ssv: get_US_TR_data.r

Larger code to download data from datamarket.com and ECB, detrend, plot and save it: getdata.r

Libre Calc/MS Excel

Useful templates; so far: Create quarterly date labels; average monthly to quarterly series. Libre Calc; MS Excel
